# Chat with Your Notes

An AI-powered PDF Q&A bot using LangChain, FAISS, and OpenAI.

## 🔧 Features
- Upload PDF files
- Ask questions
- Get contextual answers

## 🛠️ Tech Stack
- Streamlit
- LangChain
- FAISS
- OpenAI API

## 🖼️ UI Preview
![Dashboard](images/chat_with_notes_dashboard.png)

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
